# Тестовое задание: Верстка сайта Wellnkey

## О проекте
Адаптивная верстка интернет-магазина Wellnkey по предоставленному макету.

## Технологии
- HTML5
- CSS3
- Flexbox/Grid
- Адаптивный дизайн
- SVG изображения

## Особенности
- Полная адаптивность (320px - 1600px+)
- Семантическая разметка
- CSS переменные для цветов и отступов
- Оптимизированные изображения
- Кроссбраузерная совместимость

## Структура проекта
/
├── index.html          # Основной HTML файл
├── styles.css          # Основные стили
├── responsive.css      # Адаптивные стили
├── images/             # Изображения
└── README.txt          # Документация

## Запуск
1. Скачайте архив
2. Распакуйте
3. Откройте index.html в браузер / используя расширение Live server в VS Code
